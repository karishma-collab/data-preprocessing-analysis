# Data Cleaning and Analysis Project

## 📌 Overview
This project focuses on cleaning and analyzing datasets like Loan and Titanic dataset.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Features
- Handling missing values
- Removing outliers
- Data analysis and visualization

## 📁 Dataset
- loan.csv
- titanic.csv

## 🚀 How to Run
1. Install dependencies:
   pip install pandas numpy matplotlib seaborn

2. Run the script:
   python dataCleaning/dataClean6.py
